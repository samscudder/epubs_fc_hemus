# Júpiter à venda

## Correções do original

### Uma abelha se importa, página 128

Está faltando e qual número de 1958 de Future foi publicado o "Pobres Imbecis". Foi completado com a informação correta (Fevereiro de 1958).

### Pobres imbecis, página 129

No segundo paragrafo, está escrito "... continha a lista das numerosas **corridas** através das galáxias..." No original temos "... contained the list of the numerous **races** throughout the galaxies... ". Deveria ser "raças" ao invés de "corridas".

### Uma estátua para papai, página 144

No texto original fala "... **No one** who has once tasted...", mas foi traduzido como "... **Qualquer pessoa** que já tenha provado...". Deveria ser "**ninguém**" ou "**nenhuma pessoa**" para fazer sentido.
