# Nós, Marcianos

## Correções

Na página 51 (“Nós, Marcianos...” Capítulo 7) do livro, o segundo parágrafo está assim:

> A nave saiu do burado raspando e depois subiu uma milha para cima, antes que Swenson conseguisse pará-la.

O original em inglês é:

> The ship scraped up from the pit, then shot into space half a mile before forward jets could bring it to a halt.

Como não existe no dicionário a palavra "burado", acredito que seja "buraco".
